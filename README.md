# Russian Toxicity Classifier API 
 
"FastAPI service for Russian text toxicity classification" 
 
"## Features" 
"- GET /health - Service status" 
"- POST /predict - Single text classification" 
"- POST /predict_batch - Batch classification" 
"- GET /model_info - Model information" 
 
"## Usage" 
"uvicorn app.main:app --reload" 
 
"## Testing" 
"pytest"
